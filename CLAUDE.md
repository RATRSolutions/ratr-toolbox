# CLAUDE.md – Verktøykasse

_v1.2 – 2026-04-16_

## Prosjekt

En web-basert applikasjon som samler nyttige verktøy i én felles verktøykasse.

## Språk

- Forklaringer og samtale: norsk
- Kode, variabelnavn, kommentarer i filer, git commits: engelsk

## Arbeidsmetode

- Forklar hva og hvorfor før du utfører – ikke bare gjør
- Spør før store endringer
- Nevn alternativer kort når det finnes flere måter å løse noe på
- Påpek hvis noe avviker fra planen i `docs/project-status.md`
- Ved fullført steg eller viktig beslutning: foreslå oppdatering av relevant fil i docs/, men spør først

## Dokumentasjon

Les kun det som er relevant for sesjonen:

| Fil | Innhold | Les når |
|-----|---------|---------|
| `docs/project-status.md` | Nåværende fase, faseliste, neste steg | Planlegge neste steg, sjekke fremdrift |
| `docs/project-log.md` | Kronologisk hendelseslogg | Se tilbake på hva som ble gjort og hvorfor |
| `docs/architecture.md` | Teknologivalg, mappestruktur, konvensjoner | Opprette filer, diskutere struktur eller teknologi |
| `docs/spec.md` | Spesifikasjon, krav, scope | Implementere funksjoner, vurdere scope |
| `docs/dev-guide.md` | Generisk oppsettguide, gjenbrukbar til nye prosjekter | Se hvordan noe ble gjort, starte nytt prosjekt |
| `docs/verktøykasse-prosjekt.md` | Prosjektbeskrivelse og valgte prinsipper | Orientering, onboarding, prosjektspesifikke beslutninger |
